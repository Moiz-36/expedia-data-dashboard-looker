# Expedia Data Dashboard (Looker)

## Overview
This project is an interactive dashboard built using Looker to analyze booking behavior and user activity on a travel platform dataset.

## Key Metrics
- Total platform searches
- Confirmed bookings
- Booking conversion (click → booking)
- Average distance traveled
- Flights + hotel package usage

## Analysis Performed
- Compared mobile vs desktop user traffic
- Analyzed booking trends over time
- Evaluated conversion rates between clicks and bookings
- Explored travel party composition across destinations

## Key Insights
- Desktop users account for the majority of traffic (~85%)
- There is a visible drop-off between clicks and confirmed bookings
- Booking activity varies across time periods

## Tools Used
- Looker Studio
- Data visualization
- Structured dataset analysis
